# 低代码开发框架Dify - 作业说明

## 一、简介

本作业要求同学们尝试使用低代码开发框架Dify来搭建一个应用。

本次作业主要是体验性质的，同学们可以记录下搭建应用的过程以及心得体会。

## 二、作业要求

同学们需要搭建一个Chatflow来实现自己想要的功能，你可以利用插件市场提供的插件来扩展自己的Agent能力，以下是一些参考选题：

- 课程PPT问答小助手：搭建自己的知识库，然后依赖这些知识库的问答助手
- RSS订阅小助手：利用插件订阅当前感兴趣的消息，再使用大模型进行内容汇总等
- ......

这个能做的东西很多很多，你甚至可以接入豆包图片生成模型、MySQL数据库、绘图等插件扩展Agent的能力，发挥自己的想象力。

## 三、作业指导

Dify的安装可以参考Github官方文档：https://github.com/langgenius/dify

建议使用Docker来一键启动。

Dify的官方教程文档：https://docs.dify.ai/zh/use-dify/getting-started/introduction

大家遇到困难可以查阅文档。

## 四、作业提交

同学们需要提交一个文档，描述自己实现的功能、工作流设计以及最终实现的效果，期待大家的想象力。
