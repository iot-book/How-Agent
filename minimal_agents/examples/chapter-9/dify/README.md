# 第九章示例：Dify 工作流

这一目录对应教程中的“第九章 低代码框架 Dify”，用于存放本章中提到的实际工作流导出文件。

## 目录说明

- `知识库路由.yml`
  演示如何根据查询内容选择不同知识库进行检索。
- `高级知识检索.yml`
  演示如何在工作流中加入高级检索策略。
- `格式化输入输出工作流.yml`
  演示如何把输入格式化、知识检索与输出格式校验串成一个可复用工作流。
- `出题小助手.yml`
  一个面向具体任务的小型工作流示例。

## 使用方式

这些文件可以在 Dify 中直接导入，作为工作流示例进行查看、修改与复用。

建议配合教程正文一起阅读：

1. 先在教程中理解每个工作流的作用；
2. 再导入对应的 `.yml` 文件查看节点配置；
3. 最后根据自己的需求继续修改。
