# 第八章示例：记忆与检索

这一目录对应教程中的“第八章 记忆与检索”，用于展示一个最小但完整的记忆系统与 RAG 检索原型。

## 目录说明

- `examples/demo.py`
  一个最小演示脚本，用于串起记忆模块与检索模块的基本调用方式。
- `src/memory/`
  记忆系统实现，包括工作记忆、情景记忆、语义记忆以及统一管理器。
- `src/rag/`
  检索增强生成相关实现，包括文本分块、检索与候选结果组织。
- `src/utils/`
  记忆系统与检索系统共用的工具函数。

## 阅读建议

建议先阅读教程中的概念讲解，再对照这一目录查看代码结构：

1. 先看 `src/memory/models.py` 理解数据结构；
2. 再看 `src/memory/types/` 理解三类记忆的差异；
3. 然后看 `src/memory/manager.py` 理解统一调度；
4. 最后看 `src/rag/manager.py` 理解检索部分如何接入。

## 运行方式

可以先从最小示例开始：

```bash
python minimal_agents/examples/chapter-8/memory/examples/demo.py
```

这一目录的作用主要是帮助读者理解“记忆系统与检索系统如何拆分模块并组织代码”，不要求一开始就接入真实模型或真实向量数据库。
